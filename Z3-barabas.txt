- style.css - CSS súbor so štılmi pre prezentáciu
- transformation.xsl - xsl súbor pre tranformáciu do formátu XHTML 

Spôsob prekladu do formátu XHTML:
- cez príkazovı riadok dosta sa do prieèinka projektu
- príkazom java -jar saxon.jar -xsl:transformation.xsl -s:presentation.xml sa do prieèinka output vygenerujú XHTML slajdy
- monos poui parametre page-number a font-type nasledovnım spôsobom: java -jar saxon.jar -xsl:transformation.xsl -s:presentation.xml page-number=false font-type=Verdana
- keïe sa pouíva XSLT 2.0 treba poui SAXON od verzie 8.0 vyššie
